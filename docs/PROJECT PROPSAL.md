## RehabPlay: Gamified Physiotherapy Through Motion Tracking 

## Project Overview 

RehabPlay is an interactive rehabilitation platform that transforms physiotherapy exercises into engaging motion-based activities using computer vision. Instead of performing repetitive and often boring rehabilitation exercises, users can complete therapeutic movements through simple games and interactive challenges. 

The system uses a standard webcam to detect body movements in real time and provides immediate visual feedback, making rehabilitation more enjoyable and motivating. 

## Problem Statement 

Physiotherapy is essential for recovery among stroke patients, elderly individuals, and people with movement-related conditions. However, many patients struggle to remain consistent with prescribed exercises because the rehabilitation process can be repetitive, monotonous, and difficult to track. 

Low engagement often leads to reduced exercise adherence and slower recovery. 

## Proposed Solution 

RehabPlay introduces a gamified rehabilitation experience where users perform physical movements that are recognized through computer vision and translated into game interactions. 

By combining motion tracking with interactive activities, the platform encourages users to stay engaged while performing therapeutic exercises. 

## Target Users 

- Stroke Patients 

- Elderly Individuals 

- Parkinson’s Patients 

- Physiotherapy Students (for demonstration and training purposes) 

## Core Features 

## **1. Real-Time Motion Tracking** 

The system uses a webcam to track body movements and identify user gestures without requiring any special hardware. 

## **2. Interactive Rehabilitation Activities** 

## _Butterfly Catch_ 

Users move their hands to catch flying butterflies appearing on the screen. 

## _Fruit Reach_ 

Users stretch their arms to reach and collect fruits appearing at different positions. 

## _Arm Raise Exercise_ 

Users perform controlled arm-raising movements while the system automatically counts repetitions. 

## **3. Performance Feedback** 

After each activity, users receive: 

- Completion Score 

- Accuracy Percentage 

- Number of Successful Actions 

- Exercise Repetitions 

## **4. Engaging User Interface** 

The platform includes modern animations, motion graphics, and interactive visual effects to create an enjoyable user experience. 

System Workflow 

1. User opens the application. 

2. Webcam captures live video. 

3. Motion tracking detects body landmarks. 

4. User movements control game interactions. 

5. Performance metrics are calculated. 

6. Results are displayed at the end of each session.
